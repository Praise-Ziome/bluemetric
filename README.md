# BlueMetrics

BlueMetrics is a simple web-based monitoring system built with PHP, MySQL, HTML, and CSS.  
This project allows users to log in, view monitoring dashboards, and manage sessions.

## Features
- User authentication (login/logout)
- Password hashing and verification
- Dashboard interface
- Simple responsive design
- Prepared for further features (analytics, alerts, etc.)

## Requirements
- XAMPP or any local PHP/MySQL server
- PHP 7.4+ (tested on XAMPP)
- MySQL database with a `users` table
- Modern web browser
